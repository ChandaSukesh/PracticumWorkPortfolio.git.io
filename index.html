MSIT Dashboard

The main goal of this project is to make more data accessible for the students. Both Mentors and Students have privileges to login into the Dashboard. The students can see their Zoom attendance, Course grades, Mock Interviews data, Competitive Programming Scores. Moreover they can visualize their performance in the curriculum by observing Line Graphs or Bar Graphs. Mentors can see the data of any student with the feature of most frequently searched students name appearing on the Search Bar.

Technologies Used


React

For MSIT Dashboard we have used React Js as a front end technology. We have implemented various components like Jumbotron, Accordian, Tables and made them responsive.
Also, we implemented the video component using the React-player. The React-player is a react component for playing a variety of URL's, and we can play them in YouTube as well.
Moreover, we implemented Bar Graphs and Line Graphs using the Chart-js Library. The Chart-js library is a open source library that helps us to easily visualize data using JavaScript. With the help of this library we can implement graphs from it and all of them would be responsive.

Flask

Flux Advanced Security Kernel is an API of Python that allows us to build up web-applications. So, to design MSIT Dashboard we have used Server side Flask as our backend. Moreover, we have used flask for reading the student's data from the CSV file. Here, we have read the CSV raw data and converted it into the required json format. And we presented the same on website using React.
